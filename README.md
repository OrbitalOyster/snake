# Snake Game

## Prerequisites

* git
* cmake
* clang
* This: https://github.com/libsdl-org/SDL/blob/main/docs/README-linux.md#build-dependencies

## Downloading, building and running

```bash
git clone https://github.com/orbitaloyster/snake.git --recursive
cd snake
cmake . -B build
make -C build
./build/main
```
